# Wavelength

A private, non-commercial, pass-and-play web game inspired by Wavelength.

## Files

- `index.html` opens the game.
- `styles.css` contains the visual design.
- `script.js` contains the game logic and demo clue cards.
- `wavelength-odometer-icon.png` is used as the browser icon.
- `wavelength-odometer-icon.ico` is used by the local Windows shortcut.

## GitHub Pages

This project is a static website. Upload the files to a GitHub repository, then enable GitHub Pages from the repository settings using the `main` branch and root folder.

## Custom Cards

The temporary demo spectrum cards live near the top of `script.js` in the `demoCards` array. Replace those entries later with the final custom binaries.
